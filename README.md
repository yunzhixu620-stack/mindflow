# MindFlow - 实习生的思维笔记 🦞

> AI 驱动的思维导图工具，专为无领导小组会议、需求讨论会和知识整理设计。

**在线 Demo:** https://yunzhixu620-stack.github.io/mindflow/

---

## ✨ 核心功能

### 🎯 会议助手
- **无领导小组讨论** - 自动记录观点、整理讨论框架
- **需求评审会** - 提取需求点、优先级、排期建议
- **发散讨论** - AI 实时提供思维碰撞，拓展思考维度

### 💡 知识碎片
- 输入零散知识点，AI 自动整合成结构化知识树
- 支持中英文混合（适合阅读论文）
- 持续学习你的知识库，越用越聪明

### 📄 文章解析
- 粘贴长文/公众号/论文，AI 提炼核心论点
- 自动生成思维导图，清晰展示逻辑链条
- 多篇文档关联，构建知识网络

## 🚀 快速开始

### 1. 打开网页
访问 https://yunzhixu620-stack.github.io/mindflow/

### 2. 配置 API（首次使用）
点击右上角 **设置** → 填入：
- **API 代理地址**: `https://mindflow-proxy-dbyxpormng.cn-hangzhou.fcapp.run`
- **API Key**: （可选，如果代理内置了 Key）
- **AI 模型**: 推荐 `Qwen Plus`

⚠️ **安全提醒：** 如果你是自己部署的代理，请参考 [DEPLOY.md](./DEPLOY.md) 设置环境变量，不要硬编码 API Key！

### 3. 开始使用
- **会议模式**: 点击"发散讨论"，输入会议目标如"如何提升用户留存率"
- **知识模式**: 直接输入知识点，如"费曼学习法：用简单语言解释概念..."
- **文章模式**: 粘贴文章全文，AI 自动提炼

## 🛠️ 技术栈

- **部署**: GitHub Pages + 阿里云函数计算
- **存储**: LocalStorage (本地) + GitHub Gist (即将支持)
